## Getting Started

### Prerequisites

* Python 3.x installed
* pip package manager

### Installation

1. Clone the repo

   ```bash
   git clone https://github.com/your-username/digital-library.git
   cd digital-library
   ```

2. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

3. Run the app

   ```bash
   flask run
   ```

4. Open your browser and go to

   ```
   http://127.0.0.1:5000/
   ```
